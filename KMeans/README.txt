K-MEANS clustering on images:
----------------------------

Open the command prompt and go to the folder where you store the file KMeans.java and execute the following command:

Step-1:
javac KMeans.java

Step-2:
java KMeans <Penguins.jpg/Koala.jpg> <K-value> <output-file.jpg>
Ex: java KMeans Koala.jpg 5 K5.jpg



Python:
------

The program was written in Python 3.6 
Go to command prompt and go to the folder where you store the file final.py and execute the following command:

Step-1:
python final.py <Penguins.jpg/Koala.jpg> <K-value> <Number of iterations>
Ex: python final.py Penguins.jpg 5 15
